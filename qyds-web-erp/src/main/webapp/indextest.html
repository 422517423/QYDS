12.ERP电商通用优惠劵发行接口
参数说明:
couponId:优惠券ID,UUID生成
couponName:优惠券名称
couponType:优惠券类型,10代金券,20生日券,30注册送券
goodsType:10.全部,20.商品分类,30.品牌,40.商品,50.SKU
distributeType:优惠券下发方式,10.平台推送,20.满购赠送,30.手动领取
isOriginPrice:满购条件是否仅限计算正价商品,0.不限,1.仅正价,2.仅折扣商品
minGoodsCount:最少商品件数,用来判断是否符合满购件数的使用条件
minOrderPrice:最小订单总价,用来判断是否符合满购金额的使用条件
worth:优惠券价值
sendStartTime:优惠券发券开始时间
sendEndTime:优惠券发券结束时间
startTime:优惠券开始时间
endTime:优惠券结束时间
maxCount:优惠券最大发放数量
validDays:优惠券有效天数,类型20,30是须指定
distributedCount:已发放数量,根据发放接口的数量计算,此处不用传,自动设为0
comment:优惠券说明
结果说明:resultCode:00,正常;99,异常
